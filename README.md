# personal-project.py
Retail Intelligence Analytics  &amp; Forecasting Dashboard.
This project analyzes retail sales data and predicts future sales using machine learning.

## Features

- Data Cleaning
- Data Wrangling
- Sales Analysis
- Interactive Dashboard
- Sales Forecasting
- Business Insights

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Streamlit
- Scikit-Learn

## Results

- Identified top-selling categories
- Analyzed city-wise performance
- Forecasted future sales

## How To Run

pip install -r requirements.txt

streamlit run src/dashboard.py
